About Us

J'PURA FOSS Community

FOSS community is a world wide community which is aeoumd the world.It was launched in Sri Lanka as foss.lk since around 2007. The main task is mainly pramoting the open source software concept via the society and the University and also to motivate the developers and make a better background to develop such FOSS softwares.

University of
Sri Jayewardenepura

University Of Sri Jayewardenepura started the education firstly in Buddhist education around 16th of February 1959 by Soratha Maha Thero. Later th University commenced it's academic activities with five faculties."Vijja Uppa thathathn settha" takes as the motto of the University. At the present University has given the education around thousands of students for the best carrier of them.

J'pura FOSS community is a team of volunteers who helps to raise awareness about FOSS among curious tech enthusiasts. Giving from basic to advanced knowledge of Free and open source software in a variety of ways is the main objective of the community.

This webinar provides an introduction to using Git and GitHub to manage, share and backup your code. We will discuss
  1. Why you should use version control
  2. What GitHub is
  3. How to get started with GitHub
  4. Additional useful commands
At the end of the webinar you should be able to organise your code into a git repository and see it on GitHub along with knowing how to edit and update the repository. This will allow you to keep track of changes and easily share your code with others.
